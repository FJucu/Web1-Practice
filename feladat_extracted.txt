 Alakítsanak ki két- fős csoportokat és a feladatot közösen, projektmunkában oldják meg. 
 A dokumentációban majd írják le, hogy a két főből ki melyik feladatrészt csinálta meg. 
 Kötelező csoportban dolgozni! (A Teams csoportban is írhatnak üzenetet egymásnak, hogy társat keresnek a feladathoz) 
 Ha valaki nem talál társat így sem , akkor írjon a tanárnak üzenetet. A csoportmunka célja : a cégeknél is csoportban dolgoznak egy nagyobb feladaton, ezt gyakorolják itt is. 
 Ha a csoportban páratlan számú hallgató készít beadandót, akkor a tanár annak engedélyezi, hogy egyedül csinálja a feladatot. 
 Ebben az esetben, ha meg akarja kapni a GitHub-o s projektmunka feladatra is a pontot, akkor hozzon létre két GitHub fiókot és azokon valósítsa meg megoldást. 
 A párból elég, ha csak az egyik személy nevén valósítják meg az alkalmazást az Interneten.

 Készítsen egy web-alkalmazást a következő feladatok megvalósítására: Főoldal menü (index.html) : Mutassa be az alkalmazást látványos weboldalon. ( 2 pont) Fejlécbe írassa ki H1-es címsorral : Web programozás-1 El őadás Házi feladat Láblécbe írja be a készítők nevét és Neptun kódját. JavaScript menü (javascript.html) : ( 2 pont) Készítsen egy CRUD alkalmazást JavaScript-el, amihez használja fel a választott adatbázis egyik fájljának adatait. Az alkalmazás adatait egy tömbben tárolja. React menü ( react .html) : ( 2 pont) Készítsen egy CRUD alkalmazást React -el, amihez használja fel a választott adatbázis egyik fájljának adatait. SPA menü (spa.html): ( 3 pont) Készítsen egy Egyoldalas alkalmazást ( Single page application , SPA) 2 menüvel. A két menüpontnál valósítson meg 1-1 szabadon választott React alkalmazást, amelyeknek a nehézségi foka legyen hasonló, mint az órán bemutatott Calculator , Tic-Tac-Toe játék alkalmazások nehézségi foka. Mindkettőben használjon komponenseket és useState állapotváltozókat. A dokumentációban adja meg, hogy honnan használta fel a két React alkalmazást. A GitHub- on készítsen egy react mappát, amibe tegye fel a két alkalmazás src és dist mappáinak a tartalmát. Fetch API menü (fetchapi.html): ( 4 pont) Készítsen egy CRUD alkalmazást JavaScript-el és Fetch API- val , amihez használja fel a választott adatbázis egyik fájljának adatait. Az alkalmazás adatait a szerver oldalon adatbázisban tárolja . Axios menü (axios.html) ( 4 pont) Készítsen egy CRUD alkalmazást React -el és Axios-al , amihez használja fel a választott adatbázis egyik fájljának adatait. Az alkalmazás adatait a szerver oldalon adatbázisban tárolja. OOJS menü (oojs.html) ( 3 pont) Készítsen egy szabadon választott Objektumorientált JavaScript grafikus alkalmazást . Alkalmazza a következőket: class , constructor , metódusok, document.body .appendChild , extends , super Alkalmazását töltse fel és valósítsa meg Internetes tárhelyen is.
